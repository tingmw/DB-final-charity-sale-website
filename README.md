# Online Charity Sale System

This is a web application for an online charity sale system, built with Python Flask and SQLite.

## How to Run

1. **Install dependencies:**
   ```bash
   pip install Flask markdown
   ```

2. **Initialize the database:**
   ```bash
   python init_db.py
   ```

3. **Run the application:**
   ```bash
   python app.py
   ```

4. **Access the website:**
   Open your web browser and navigate to `http://127.0.0.1:5000`.
